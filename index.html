Objetos:

Un Objeto es una entidad de la vida real abstraida en código

          
          const koderObject = {
            name: 'Fernanda',
            lastName: 'Palacios'
            generation: '11',
            birthday: '1996/06/24',
          }

          const personObject = {
            name: 'Abril',
            lastName: 'Palacios',
            birthday: '2000/03/30',
            hobbies: [
              'Leer',
              'Ver anime',
              'Tocar la guitarra'
            ]
          }

Acceder a sus propiedades

          
          console.log(koderObject.name) // 'Fernanda'
          console.log(koderObject['name']) // 'Fernanda'

Agregar propiedades

          
          koderObject.bootcamp = 'JavaScript'
          /*
            {
              name: 'Fernanda',
              lastName: 'Palacios'
              generation: '11',
              birthday: '1996/06/24',

            }
          */
          koderObject['bootcamp'] = 'JavaScript'
          /*

            lastName: 'Palacios'
            generation: '11',
            birthday: '1996/06/24',
            bootcamp: 'JavaScript'
          }
          */

Constructor Function:

Nos permitirá crear Tipos de Objetos, donde podemos definir sus propiedades y métodos

Declarar una Función Constructora

          
          function Koder(name, lastName, generation){
            this.name = name
            this.lastName = lastName
            this.generation = generation
            this.greet = function(){
              console.log(`Hola mi nombre es ${this.name} y soy de la ${this.generation}º Generation`)
            }
          }

Instanciar

          
          const koder = new Koder('Juan Pablo', 'Sánchez', 10)
          console.log(koder.name) // Juan Pablo
          koder.greet() // Hola mi nombre es Juan Pablo y soy de la 10º Generation

¿Qué es This?

Palabra reservada que hace referencia al objeto actual
